DIJKSTRA SEGMENTATION README

NOTE: Requires the 'Image Processing Toolbox'.


The script 'dijkstra_segmentation_script.m' runs through the entire
segmentation pipeline using one of the VEVIO frames.

The main segmentation file is 'dijsktra_segmentation.m'. It expects a
filtered image, such as that produced by 'filter_image.m'.
